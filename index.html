Projects

DJ 183

Given only a few weeks to do so, I lead a team of 4 to create an online music quizzer. Wary of how important teamwork and communication is, I created a groupchat to communicate on and set up meetings at least twice a week. Once everyone knew what to work on, we set off to create the best website possible. I took on the role of styling the website using bootstrap, CSS keyframing, and general CSS styling properties. The team worked on using Flask to pull a preview of a song from a billboard API, give the user 4 options to choose from to try and then have the user guess the song that was played. Flask was also used to keep track of the user's score. Regardless of whether the user got the answer right or wrong, the lyrics of the correct song were displayed after the user made their choice through the use of beautifulsoup. Lastly, the team used google firebase to create a leaderboard of scores.
